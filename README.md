 # วิธีติดตั้ง

เลือก file path ที่ต้องการจะเก็บไฟล์ไว้
```
cd filepath
```

clone repo
``` 
git clone https://github.com/Timemi11/FrontEnd_botnoi_piramid.git
```

เปลี่ยน directory ไปที่ FrontEnd_botnoi_piramid
```
cd FrontEnd_botnoi_piramid
```

ติดตั้ง dependency
```
npm i
```
start server
```
npm run dev
```
